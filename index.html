📚 产品理念

个人阅读与思考助手，帮助用户把阅读过程转化为长期积累的认知资产。

核心流程：

阅读 → 摘录 → 思考 → 分类 → 沉淀

✨ 功能介绍

📖 阅读记录： 保存阅读标题、来源、标签和时间。

✍ 思考记录： 保存个人理解和观点。

📚 阅读档案： 形成属于自己的知识库。

📝 Markdown导出： 方便保存和整理。

📌 使用流程

1. 开始阅读，记录阅读过程。

2. 保存文章摘录和自己的思考。

3. 在阅读档案中查看长期积累。

4. 导出内容，保存个人知识资产。

🌱 写在最后，我有一些心里话

这个项目的诞生，源于一个很简单的问题：

我们每天阅读大量信息，但真正留下来的内容有多少？

我希望通过这个工具，让阅读不只是一次信息输入，而是一次思考、整理和积累的过程。

这个项目也是我第一次尝试独立完成一个软件产品。 没有专业的计算机背景，也没有电脑，只是在不断学习、测试和修改中，一步一步完成。

AI可以帮助我们学习和创造，但真正属于自己的理解、思考和认知，仍然需要自己建立。

希望这个工具可以帮助更多人，让每一次阅读都留下痕迹，让每一次思考都成为成长的积累。
